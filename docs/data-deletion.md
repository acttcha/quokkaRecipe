# 쿼카레시피 데이터 삭제 안내

**최종 업데이트: 2026년 7월 19일**

쿼카레시피(이하 "본 앱")는 사용자가 자신의 데이터를 언제든 삭제할 수 있도록 다음 방법을 제공합니다.

---

## 1. 삭제할 수 있는 데이터의 종류

| 구분 | 저장 위치 | 예시 |
| --- | --- | --- |
| 기기 내 데이터 | 사용자 기기(보안 저장소) | 저장한 레시피·폴더·메모, 냉장고 재료, 식이 선호도, Q&A 기록, 요리 일기 |
| 서버 데이터 | 클라우드 서버(Supabase) | 잎사귀 잔액·사용 내역, 앱 사용 통계, (로그인 시) 계정 정보(이메일 등) |

---

## 2. 기기 내 데이터 삭제 방법

### 방법 1: 앱 내 메뉴에서 개별 삭제

- **냉장고 재료**: 냉장고 탭에서 "비우기" 또는 개별 항목 삭제
- **저장한 레시피**: 레시피 탭에서 개별 삭제
- **메모 및 Q&A 기록**: 레시피 상세 화면에서 삭제
- **요리 일기**: 요리 일기 화면에서 개별 삭제

### 방법 2: 모든 데이터 한 번에 삭제 (앱 내)

- **마이 탭 → "모든 데이터 삭제"** 를 선택하면 기기에 저장된 모든 로컬 데이터가 즉시 삭제됩니다. (확인 문구 입력 필요)

### 방법 3: 앱 삭제 (전체 삭제)

앱을 기기에서 삭제하면 **모든 로컬 데이터가 즉시 영구 삭제**됩니다.

- Android: 홈 화면에서 앱 아이콘 길게 누르기 → 삭제(또는 제거)
- iOS: 홈 화면에서 앱 아이콘 길게 누르기 → 앱 삭제

추가 보관 기간: **없음** (앱 삭제 즉시 완전 삭제)

---

## 3. 로그아웃 (기기 연결 해제)

- 앱 내 **마이 탭 → 프로필(상단 카드) → 로그아웃** 을 누르면 이 기기와 계정의 연결이 해제됩니다. (계정 자체는 삭제되지 않으며, 계정 잎사귀는 다시 로그인하면 그대로 나타납니다.)

---

## 4. 계정 및 서버 데이터 완전 삭제

로그인 계정과 서버에 저장된 데이터(잎사귀 잔액·사용 내역, 계정 정보)를 완전히 삭제합니다.

### 방법 1: 앱에서 직접 (즉시)

- 앱 실행 → **마이 탭 → 프로필(상단 카드) → 계정 삭제** 를 선택하면 계정과 서버 데이터가 **즉시 영구 삭제**됩니다. (확인 문구 입력 필요, 되돌릴 수 없음)

### 방법 2: 이메일 요청 (앱에 접근할 수 없는 경우)

앱을 이미 삭제했거나 앱에 접근할 수 없는 경우 아래로 요청해 주세요.

- **이메일**: chasoft.official@gmail.com
- **제목**: [계정 삭제 요청]
- **본문에 포함할 내용**: 로그인에 사용한 구글 계정 이메일

요청을 받으면 본인 확인 후 지체 없이(영업일 기준 최대 7일 이내) 삭제합니다.

## 5. 외부 서비스로 전송된 데이터

앱 사용 중 일부 정보가 다음 외부 서비스에 전송됩니다. 각 서비스의 데이터는 해당 서비스의 보관 정책을 따르며, 삭제를 원하시는 경우 각 서비스에 직접 요청하실 수 있습니다.

- **Google (Gemini AI, YouTube, 로그인, AdMob)**: 식재료/영수증 이미지·입력 텍스트(AI), 검색어(YouTube), 계정 정보(로그인), 광고 식별자(AdMob)
    - 보관 정책: https://policies.google.com/privacy
- **Supabase**: 잎사귀 잔액·사용 내역, 계정 정보 — 위 4항(계정 삭제)으로 삭제됩니다.
    - 보관 정책: https://supabase.com/privacy
- **RevenueCat**: 구매·구독 내역
    - 보관 정책: https://www.revenuecat.com/privacy
- **쿠팡 파트너스**: 재료명 (구매 링크 클릭 시)
    - 보관 정책: https://www.coupang.com/np/privacy

> 참고: AI(Gemini) 요청 내용은 회사 서버에 별도로 저장되지 않습니다. 인앱 결제·구독 내역은 Google Play 및 RevenueCat에도 기록되며, 구독 해지는 Google Play의 구독 관리 화면에서 진행해야 합니다.

## 6. 삭제되는 데이터와 보관되는 데이터

- **삭제**: 잎사귀 잔액·사용 내역, 계정 정보(이메일 등), 앱 사용 통계, 기기 내 모든 로컬 데이터
- **보관**: 관련 법령(전자상거래법 등)에 따라 보관 의무가 있는 결제·거래 기록은 해당 법정 기간 동안 보관될 수 있습니다. 이 경우 해당 정보는 삭제 목적 외로 이용되지 않습니다.
