# 쿼카레시피 개인정보처리방침

**최종 업데이트: 2026년 7월 19일**

쿼카레시피(이하 "본 앱")는 사용자의 개인정보를 소중히 다루며, 「개인정보 보호법」을 비롯한 관련 법령을 준수합니다. 본 방침은 본 앱이 수집·이용·제공하는 정보의 종류와 그 처리 방법에 대해 설명합니다.

본 앱은 **로그인 없이 게스트로 이용**할 수 있으며, 기기 변경·재설치 시 잎사귀 잔액을 이어서 사용하려는 경우에 한해 **선택적으로 구글 로그인**을 제공합니다.

---

## 1. 수집하는 정보

### 1.1 사용자가 직접 입력·제공하는 정보

- 식재료 사진 (카메라 촬영 또는 사진 라이브러리에서 선택)
- 영수증 사진 (사용 시)
- 직접 입력한 재료 이름
- 식이 선호도 (알레르기, 매운맛 선호도, 조리 시간, 식단 유형, 요리 실력, 선호 음식 스타일)
- 레시피에 대한 메모 및 질문 내용
- 요리 일기용으로 직접 촬영한 요리 사진
- **(선택) 구글 로그인 시**: 구글 계정 이메일, 프로필(이름·사진), 계정 식별자

### 1.2 자동으로 수집되는 정보

- 기기 식별자 — 로그인하지 않은 게스트의 잎사귀 잔액을 기기와 연결하기 위한 식별자
- 앱 사용 통계 — AI 호출의 종류·횟수·시각, 재료 스캔 횟수 등
- 잎사귀 잔액 및 사용(차감·적립) 내역
- 저장한 레시피 정보 (사용자가 ♥ 버튼으로 저장한 경우)
- (광고 표시 시) 광고 식별자 및 광고 관련 정보 — 아래 3.6 참조

### 1.3 수집하지 않는 정보

- 비밀번호 — 본 앱은 구글 로그인만 지원하며 비밀번호를 저장하지 않습니다
- 위치 정보
- 연락처, 통화 기록 등 기기 내 다른 앱의 정보

## 2. 정보의 저장 위치

본 앱의 정보는 저장 위치에 따라 다음과 같이 나뉩니다.

### 2.1 기기 내 저장

- 식이 선호도
- 저장한 레시피, 폴더, 메모
- 쿼카 Q&A 기록
- 냉장고 재료 목록
- 요리 일기(직접 촬영한 요리 사진)

위 정보는 사용자의 기기 내 보안 저장소(iOS Keychain, Android KeyStore)에 저장되며, 회사(차소프트)의 서버로 전송·저장되지 않습니다.

### 2.2 서버 저장 (Supabase)

아래 정보는 안정적인 잎사귀 관리와 부정 이용 방지를 위해 회사가 이용하는 클라우드 서버(Supabase)에 저장됩니다.

- 잎사귀 잔액 및 사용(차감·적립) 내역
- AI 호출 사용 통계 (종류·횟수·시각)
- 기기 식별자 또는 (로그인 시) 구글 계정 정보(이메일 등)

이 정보는 **기기 식별자** 또는 **구글 계정 ID**를 기준으로 저장되어, 같은 기기에서 앱을 재설치하거나 다른 기기에서 로그인할 때 잎사귀 잔액이 유지되도록 합니다.

## 3. 외부 서비스로의 정보 제공

본 앱은 핵심 기능 제공을 위해 다음 외부 서비스를 이용합니다.

### 3.1 Google Gemini API (AI)

- **전송되는 정보**: 촬영한 식재료/영수증 이미지, 입력한 재료 텍스트, 식이 선호도, 레시피 정보, 사용자가 입력한 질문
- **이용 목적**: 이미지에서 식재료 인식, 레시피 생성, 레시피 관련 Q&A 응답, 유튜브 영상의 자막·설명 분석
- **전송 방식**: 회사가 운영하는 서버(Supabase)를 경유하여 Google로 전송되며, 회사는 해당 요청 내용(이미지·텍스트)을 서버에 저장하지 않습니다.
- **제공 처**: Google LLC (미국)
- **개인정보 처리 정책**: https://policies.google.com/privacy

### 3.2 Supabase (서버 인프라)

- **저장·처리되는 정보**: 잎사귀 잔액·사용 내역, 앱 사용 통계, 기기 식별자, (로그인 시) 계정 정보
- **이용 목적**: 잎사귀 지갑 관리, 사용자 인증(로그인), AI 요청 중계
- **제공 처**: Supabase, Inc. (미국)
- **개인정보 처리 정책**: https://supabase.com/privacy

### 3.3 Google 로그인 (선택)

- **수집되는 정보**: 구글 계정 이메일, 프로필(이름·사진), 계정 식별자
- **이용 목적**: 선택적 로그인 — 기기 변경·재설치 시에도 잎사귀 잔액을 이어서 사용
- **제공 처**: Google LLC (미국)
- **개인정보 처리 정책**: https://policies.google.com/privacy

### 3.4 RevenueCat (인앱 결제)

- **전송되는 정보**: 구매·구독 내역, 사용자 식별자(기기 식별자 또는 계정 ID)
- **이용 목적**: 인앱 결제·구독 상태 관리 및 검증
- **제공 처**: RevenueCat, Inc. (미국)
- **개인정보 처리 정책**: https://www.revenuecat.com/privacy

### 3.5 Google YouTube Data API

- **전송되는 정보**: 재료명 기반 검색어
- **이용 목적**: 레시피 영상 검색
- **제공 처**: Google LLC (미국)
- **개인정보 처리 정책**: https://policies.google.com/privacy

### 3.6 Google AdMob (광고)

- **수집·이용되는 정보**: 광고 식별자, 기기 정보, 광고 노출·클릭 등 광고 관련 정보
- **이용 목적**: 무료 이용자에게 배너·보상형 광고 제공
- **제공 처**: Google LLC (미국)
- **개인정보 처리 정책**: https://policies.google.com/technologies/ads
- 쿼카 패스(구독) 이용자에게는 광고가 표시되지 않습니다.

### 3.7 쿠팡 파트너스

- **전송되는 정보**: 재료명 (사용자가 "쿠팡에서 구매하기" 링크를 클릭한 경우에 한함)
- **이용 목적**: 재료 구매 페이지 연결
- **제공 처**: 쿠팡 주식회사
- **개인정보 처리 정책**: https://www.coupang.com/np/privacy

본 앱은 위 서비스 외에 다른 제3자에게 사용자의 정보를 제공하지 않습니다.

### 5.1 기기 내 저장 정보

- **보유 기간**: 사용자가 앱을 삭제하거나 앱 내 "모든 데이터 삭제" 기능을 사용할 때까지
- **파기 방법**: 앱 삭제 또는 초기화 시 즉시 삭제

- **보유 기간**: 잎사귀 관리 및 부정 이용 방지를 위해 보관하며, 계정 삭제 요청 시 지체 없이 파기
- **계정·데이터 삭제**: 앱 내 "마이 → 프로필 → 계정 삭제"로 계정과 서버에 저장된 데이터를 즉시 삭제할 수 있습니다. 앱에 접근할 수 없는 경우 아래 연락처(chasoft.official@gmail.com)로 요청할 수 있습니다. (자세한 절차는 "데이터 삭제 안내" 문서 참조)

각 외부 서비스의 개인정보 처리 방침에 따릅니다.

## 6. 사용자의 권리

사용자는 본 앱에서 다음 권리를 행사할 수 있습니다.

- **열람**: 앱 내 "냉장고", "저장된 레시피", "마이" 메뉴에서 저장된 정보 확인
- **수정**: 앱 내 해당 메뉴에서 직접 수정
- **삭제**: 개별 데이터 삭제 또는 앱 내 "모든 데이터 삭제"를 통한 일괄 삭제
- **로그인/로그아웃**: 앱 내 "마이" 메뉴에서 구글 계정 로그인·로그아웃
- **계정 삭제**: 앱 내 "마이 → 프로필 → 계정 삭제"로 즉시 삭제 (또는 위 연락처로 요청)
- **앱 제거**: 앱을 기기에서 삭제하면 모든 로컬 저장 데이터가 함께 삭제됩니다.

## 8. 정보 보안

본 앱은 사용자 정보를 보호하기 위해 다음 조치를 취합니다.

- 기기 내 데이터는 운영체제가 제공하는 보안 저장소(iOS Keychain, Android KeyStore)에 저장
- 서버(Supabase) 데이터는 접근 제어(행 수준 보안, RLS)로 보호되며, 앱은 서버 데이터에 직접 접근할 수 없고 검증된 서버 함수를 통해서만 처리됨
- 외부 서비스와의 통신은 HTTPS 암호화 사용
- 로그인은 구글 인증(OAuth)을 사용하며, 본 앱은 비밀번호를 저장하지 않음
- 회사는 AI 기능 제공을 위해 경유하는 사용자의 식재료 이미지 및 입력 정보를 서버에 별도로 보관하지 않음
